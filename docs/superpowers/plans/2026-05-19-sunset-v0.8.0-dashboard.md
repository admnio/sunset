- [ ] **Step 3.2: Make JobPayload self-contained**

Rewrite the class so it no longer extends Horizon. Concrete shape:

```php
<?php

namespace Admnio\Sunset;

use Illuminate\Database\Eloquent\Model;
use Illuminate\Support\Str;

class JobPayload
{
    public string $value;
    public array $decoded;

    public function __construct(string $value)
    {
        $this->value = $value;
        $this->decoded = (array) json_decode($value, true);
    }

    public function id(): string
    {
        return (string) ($this->decoded['uuid'] ?? $this->decoded['id'] ?? '');
    }

    public function prepare($job = null): self
    {
        // Inject Sunset's enrichment fields onto the payload if not already set.
        $changed = false;
        if (! isset($this->decoded['uuid'])) {
            $this->decoded['uuid'] = (string) Str::uuid();
            $changed = true;
        }
        if (! isset($this->decoded['type'])) {
            $this->decoded['type'] = $this->resolveType($job);
            $changed = true;
        }
        if (! isset($this->decoded['tags'])) {
            $this->decoded['tags'] = $this->resolveTags($job);
            $changed = true;
        }
        if (! isset($this->decoded['pushedAt'])) {
            $this->decoded['pushedAt'] = (string) microtime(true);
            $changed = true;
        }
        if ($changed) {
            $this->value = json_encode($this->decoded);
        }
        return $this;
    }

    private function resolveType($job): string
    {
        if (is_object($job)) {
            return get_class($job);
        }
        if (is_string($job)) {
            return $job;
        }
        return $this->decoded['displayName'] ?? 'job';
    }

    private function resolveTags($job): array
    {
        if (! is_object($job)) {
            return [];
        }
        $tags = [];
        if (method_exists($job, 'tags')) {
            $tags = (array) $job->tags();
        }
        // Auto-tag Eloquent models referenced as job properties.
        try {
            $ref = new \ReflectionObject($job);
            foreach ($ref->getProperties() as $prop) {
                $prop->setAccessible(true);
                $value = $prop->getValue($job);
                if ($value instanceof Model) {
                    $tags[] = class_basename($value) . ':' . $value->getKey();
                }
            }
        } catch (\Throwable) {
            // Ignore reflection issues.
        }
        return array_values(array_unique(array_filter($tags)));
    }
}
```

The exact existing public method/property names must be preserved. Read the current file before writing this. If the existing `JobPayload` uses different method names (e.g. `tags()` returning the list directly), match those.

    /** @var \Closure|null */
    private static $authCallback = null;

    public function auth(\Closure $callback): void
    {
        self::$authCallback = $callback;
    }

    public function check($request): bool
    {
        if (self::$authCallback) {
            return (bool) (self::$authCallback)($request);
        }
        if ($this->container['config']->get('app.env') === 'local') {

        }
        return in_array($request->ip(), ['127.0.0.1', '::1'], true);
    }

<?php

namespace Admnio\Sunset\Dashboard\Http\Controllers;

use Illuminate\Http\JsonResponse;
use Illuminate\Http\Request;
use Illuminate\Routing\Controller as LaravelController;
use Inertia\Response as InertiaResponse;

abstract class Controller extends LaravelController
{
    /**
     * Return an Inertia render OR JSON-only payload depending on the request.
     *
     * The frontend hits the same URL with ?refresh=1 on each polling tick;
     * the controller returns just {"props": {...}} in that case so Vue
     * doesn't have to re-mount the page.
     */
    protected function inertiaOrJson(Request $request, string $page, array $props): InertiaResponse|JsonResponse
    {
        if ($request->query('refresh') === '1' || $request->wantsJson()) {
            return response()->json(['props' => $props]);
        }
        return \Inertia\Inertia::render($page, $props);
    }
}
```

public function retry(string $id, FailedJobRepository $store): JsonResponse
{
    $record = $store->find($id);
    if (! $record) return response()->json(['error' => 'not_found'], 404);

    // Re-dispatch via queue manager onto the original connection/queue.
    app('queue')->connection($record['connection'])
        ->pushRaw($record['payload'], $record['queue']);

    $store->recordRetry($id, (string) \Illuminate\Support\Str::uuid(), time());

    return response()->json(['retried' => true]);
}

public function retryBulk(Request $request, FailedJobRepository $store): JsonResponse
{
    $ids = (array) $request->input('ids', []);
    $out = [];
    foreach ($ids as $id) {
        $resp = $this->retry((string) $id, $store);
        $out[$id] = $resp->getData(true);
    }
    return response()->json($out);
}

public function delete(string $id, FailedJobRepository $store): JsonResponse
{
    $store->forget($id);
    return response()->json(['deleted' => true]);
}

public function deleteBulk(Request $request, FailedJobRepository $store): JsonResponse
{
    $ids = (array) $request->input('ids', []);
    foreach ($ids as $id) $store->forget((string) $id);
    return response()->json(['deleted' => count($ids)]);
}

```js
import { ref, watch } from 'vue';

const theme = ref(null); // 'light' | 'dark' | null (= follow OS)

function apply(t) {
  const root = document.documentElement;
  if (t === 'dark') root.classList.add('dark');
  else if (t === 'light') root.classList.remove('dark');
  else {
    const dark = window.matchMedia('(prefers-color-scheme: dark)').matches;
    root.classList.toggle('dark', dark);
  }
}

export function useTheme() {
  function bootstrap() {
    const stored = window.localStorage.getItem('sunset.theme');
    theme.value = stored === 'light' || stored === 'dark' ? stored : null;
    apply(theme.value);

    if (theme.value === null) {
      const mq = window.matchMedia('(prefers-color-scheme: dark)');
      mq.addEventListener('change', () => apply(null));
    }
  }

  function set(t) {
    theme.value = t;
    if (t === null) window.localStorage.removeItem('sunset.theme');
    else window.localStorage.setItem('sunset.theme', t);
    apply(t);
  }

  function toggle() {
    set(document.documentElement.classList.contains('dark') ? 'light' : 'dark');
  }

  return { theme, bootstrap, set, toggle };
}

```js
import { ref, onMounted, onBeforeUnmount } from 'vue';
import axios from 'axios';

export function usePolling(url, intervalMs = 3000) {
  const data = ref(null);
  const error = ref(null);
  let timer = null;
  let inFlight = false;

  async function tick() {
    if (inFlight) return;
    inFlight = true;
    try {
      const resp = await axios.get(url, { params: { refresh: 1 } });
      data.value = resp.data.props ?? resp.data;
      error.value = null;
    } catch (e) {
      error.value = e;
    } finally {
      inFlight = false;
    }
  }

  onMounted(() => {
    tick();
    timer = setInterval(tick, intervalMs);
  });
  onBeforeUnmount(() => { if (timer) clearInterval(timer); });

  return { data, error, refresh: tick };
}

## Dashboard

Sunset ships its own operator dashboard at `/sunset` (configurable via `sunset.dashboard.path`). Inertia + Vue 3 SPA, polling every 3 seconds.

### Install

composer require admnio/sunset

### Access

By default the dashboard is accessible only from localhost outside `local` env. Register a gate to allow specific users:

```php
use Admnio\Sunset\Facades\Sunset;

public function boot(): void
{
    Sunset::auth(fn ($request) => $request->user()?->isAdmin());
}
```

### Pages

Overview · Workload · Recent jobs · Failed jobs · Pending · Completed · Metrics · Monitoring · Rate limits · Supervisors · Batches · Health.

### Themes

Auto light/dark via `prefers-color-scheme`. User can override via the header toggle (persisted in localStorage).

### Removing Horizon

Sunset v0.8.0 no longer depends on `laravel/horizon`. The `composer.json` `require` no longer pulls Horizon. If you previously had Horizon installed only because Sunset required it, you can `composer remove laravel/horizon`.

If you want both `/horizon` (Horizon's dashboard) and `/sunset` (Sunset's) mounted in the same app, install Horizon separately. The two packages register independent bindings; if you run both `horizon` and `sunset:work` against the same queue, they'll compete for jobs — that's the consumer's responsibility.

```markdown
## From v0.7.0 to v0.8.0

v0.8.0 ships the Sunset SPA dashboard at `/sunset` AND drops `laravel/horizon` from `composer.json`. Single release.

### Breaking changes

- `composer require admnio/sunset` no longer transitively installs `laravel/horizon`. If you want Horizon side-by-side, install it explicitly.
- `Admnio\Sunset\Adapters\Horizon\*` adapter classes are deleted. Consumer code that directly referenced them must move to Sunset's native contracts under `Admnio\Sunset\Contracts\*`.
- `Admnio\Sunset\JobPayload` no longer extends `Laravel\Horizon\JobPayload`. Public surface preserved (`prepare`, `id`, `value`, `decoded`).
- `Admnio\Sunset\Repositories\SunsetWorkloadRepository` implements `Admnio\Sunset\Contracts\WorkloadRepository` (new) instead of Horizon's contract.

### Migration steps

1. `composer require admnio/sunset:^0.8`
2. `composer remove laravel/horizon` (if Sunset was the only reason you had it)
3. `php artisan sunset:install` — publishes the dashboard bundle to `public/vendor/sunset/`
4. (Optional) Register an auth gate:
   ```php
   Sunset::auth(fn ($req) => $req->user()?->isAdmin());
   ```
5. Visit `/sunset` in the browser.

### Coexistence with Horizon

If you install Horizon yourself (`composer require laravel/horizon`), both dashboards coexist: `/horizon` and `/sunset`. They register independent bindings; no conflict.

If both `php artisan horizon` and `php artisan sunset:work` are running against the same queue connection, they will compete for jobs. Pick one or partition your queues — that's your choice.
```
